Sintaxis en Python: 
indentacion: espacios o tabulaciones al inicio de una línea. delimita bloques de código. 
# Bloque de código si condición es verdadera es "if condition"

#Bloque de código si la condición es falsa es "else"

los "#" sirven para colocar comentarios en una sola línea.  y tres grupos de comillas es para comentarios de varias lineas.

las mayúsculas si hacen a la diferencia y hacen que lo que escribimos varie. 

no es necesario escribir punto y coma (;) al final de cada línea, pero si quieres escribir varias instrucciones en una misma línea
si es necesario colocar punto y coma (;)

los paréntesis se utilizan para agrupar expresiones, definir funciones y realizar llamadas a funciones.

Tipo de datos básicos:

Enteros (int): son los que no tienen parte decimal. se escriben sin ponerle comillas ni puntos. 
edad = 27
año = 2024
cantidad = 2000

Flotantes (float):son los que tienen decimales. se separa la parte entera de la decimal con un punto. 
precio= 9.99
altura= 1.70

cadenas de texto (strings): son secuencias de carácteres encerradas entre comillas simple ('...') o dobles ("...") se utilizan para representar texto
nombre = "Belen"
mensaje = 'La vida se pone heavy a veces'

Booleanos: representan a los valores de verdad (true) o (false). se usan generalmente en expresiones condicionales u operaciones lógicas. 
los valores empiezan con mayuscula (True) (False)

Variables: 
etiquetas a las que le asiganmos un valor específico. 
para declarar y asignar un valor a una variable usamos el signo '='
el nombre de la variable va a la izquierda y el valor a la derecha. (izquierda = derecha)
***Los nombres de las variables solo pueden tener letras, números, y guiones bajo. no comienzan con número.. distingue mayuscula de minuscula. 
no se pueden utilizar palabras clave reservadas de Python como nombres de variables. (ej:if else, for, while, etc)
se recomienda utilizar nombres descriptivos que indiquen claramente su propósito.
